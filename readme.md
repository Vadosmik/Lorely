# Lorely — Interactive Storytelling Platform

**Lorely** is a next-generation platform for creating and experiencing interactive fiction. Unlike traditional fanfiction sites, Lorely focuses on **choice-driven narratives**, where authors "draw" their stories using a visual logic graph.

---

## Core Vision
The goal is to bridge the gap between writing a book and developing a game. 
*   **For Authors:** A powerful "Whiteboard" editor to map out complex branching plots without coding.
*   **For Readers:** An immersive, high-performance reading experience where every choice matters.

---

## Technology Stack
*   **Backend:** Python with `FastAPI`.
*   **Frontend:** React + `React Flow`.
*   **Database:** 
    *   **PostgreSQL:** Primary storage for users, finished stories, and metadata.
    *   **Redis:** "Hot storage" for real-time session caching, autosaves, and fast editor updates.
*   **Containerization:** Docker & Docker Compose.
